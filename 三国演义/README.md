# 三国演义·儿童阅读练习 制作指南

本项目将三国演义原文改编为适合三年级小朋友的阅读练习材料。

## 目录结构

- `raw/` — 三国演义各回原文
- `study/` — 制作好的阅读练习（md格式）

## 制作思路

### 1. 选取与命名

- 从原文中选取一个完整的故事段落（不必覆盖整回，挑最精彩、最适合孩子的部分）
- 用一个生动有画面感的标题，不要直接用"第X回"（如"桃花树下的三兄弟"）

### 2. 正文写法

核心原则：**主体是原文原句，自己的话只用来衔接和铺垫。**

具体做法：

- **开头**：用2-3句自己的话交代背景，帮孩子快速进入故事（交代时代、起因），然后自然过渡到原文
- **中间**：大段保留原文原句，在段落之间插入1-2句简短的衔接语，起到承上启下的作用（如"两人正聊得痛快，第三位英雄也到了"）
- **结尾**：用1-2句自己的话收束故事，可以留个小悬念引向下一回
- 衔接语要像正常文章的叙述，不要出戏（不用"小朋友你知道吗"这类口吻）
- 原文中的诗词可以省略，保持故事节奏流畅

### 3. 选材标准

- 选故事性强、画面感强的段落（打仗、结义、斗智等）
- 避免过于血腥残忍的细节
- 人物出场、对话丰富的段落更适合孩子阅读

### 4. 语法颜色标注（color.md → HTML）

每篇文章有两个文件：
- `桃园三结义.md` — 纯文本
- `桃园三结义.html` — 带主谓宾颜色标注的 HTML 页面（从 md 生成）

#### 三种颜色

| 成分 | 颜色 | HTML class | 示例 |
|------|------|------------|------|
| 主语/代词 | 橙色 | `<span class="s">` | `<span class="s">玄德</span>` |
| 谓语动词 | 绿色 | `<span class="v">` | `<span class="v">看到</span>` |
| 宾语 | 紫色 | `<span class="o">` | `<span class="o">告示</span>` |

#### 宾语标注原则（重要）

宾语标注要**克制**，否则满屏紫色反而干扰阅读。规则如下：

1. **以句号（。）、感叹号（！）为单位划分"完整句子"**
2. **每个完整句子中最多标注一个宾语** — 主谓宾结构中最核心的那个
3. **以下宾语不标：**
   - 从句、并列补充、目的状语中的宾语（如"招募勇士保卫国家"只标"告示"，不标"勇士""国家"）
   - 引号内说话内容中的宾语（整段对话不标宾语，只保留其中的主语和谓语标注）
   - 位置补语（"传到了涿县""垂到肩膀"中的地点/方位）
4. **主语和谓语标注不受此限制**，可以正常标注多个

示例：
- "朝廷赶紧下令各地张贴**告示**，招募勇士保卫国家。" → 只标"告示"
- "他们又祭拜**天地**，宰牛设酒，邀请勇士们聚集在一起。" → 只标"天地"
- 张飞说："我有不少钱财，可以招募勇士…" → 引号内不标宾语

#### 从 color.md 生成 HTML

HTML 页面结构：
- 页面顶部有颜色图例（主语/谓语/宾语）
- 正文段落用 `<p>` 包裹，段与段之间用 `<hr>` 分隔章节
- 衔接过渡句用 `<p class="transition">` 样式（居中、灰色、斜体）
- 阅读闯关部分用 `<div class="quiz">` 包裹，不加颜色标注
- CSS 定义在 `<style>` 中，使用衬线字体，白色背景(#ffffff)

### 5. 阅读题设计

在文章末尾设置3道阅读题，要求：

- 答案必须能从文章中找到，考察是否认真阅读
- 难度适中，不能一眼就能答出，需要回到文中仔细找
- 题型多样有趣，可以用：
  - **辨认题**：给出原文描写，判断说的是哪个人物
  - **连线/匹配题**：把人物与兵器、事件等对应起来
  - **细节侦探题**：针对容易忽略的小细节提问
  - **简答题**：用自己的话回答，考察理解而非死记硬背
- 难度递进：第一题简单，第二题中等，第三题需要思考
- 避免纯填空"刘备靠卖____为生"这种过于简单的题目
